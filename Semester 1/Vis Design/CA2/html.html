Basic html.

What is html?

HTML stands for Hyper Text Markup Language. HTML is the standard markup language for creating web pages. It is used to create web pages.

HTML is only intended for creating web pages. It is not a general-purpose language.

Doctype

The doctype is the first thing that is written in an HTML document. It defines the document type, it must allways be the first thing in the document, and it only appers once per page.

Example

<!DOCTYPE html>
<html>
</html>

Metadata

Metadata is information about the document, it is not visible to the user, but it can be used by the browser to make decisions about how to display the document, its also used by search engines to index the document.

Metadata is stored in the head of the document, you can learn more about it here.

Title

<html>
    <head>
        <title> Title of the page </title>
    </head>
</html>

Descripton

<html>
    <head>
        <meta name='description' content='What even is this page?'></meta>
    </head>
</html>

Headings

The HTML spec provides a set of 6 heading tags to create diferently sized headings, you should only use the h1 tag once per page, and preferably at the top of the page.

Paragraphs

To add a paragraph to your page, you can use the p tag, it is the most basic element in HTML, its used to display text.

I am a paragraph! I contain text and thats about it. 

<html>
    <body>
        <p>I am a paragraph! I contain text and thats about it.</p>
    </body>
</html>

Lists

Lists are used to display a list of items, they can be ordered or unordered, the diferrence is that unordered lists are displayed with bullet points, while ordered lists are displayed with numbers.

Images

To add an image to your page you can use the img tag, you can also add a alt attribute to the image, this is used by search engines to index the page, it is aslo used to describe the image, incase the image dosent load, it aslo aids in accessibility.

You can also reference an external file, by setting the src attribute to the url of the file, the img tag dose not only support images but also other file types, you can find a list of file types here.
