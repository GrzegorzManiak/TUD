Basic css.

What is css?

Cascading Style Sheets are a set of rules that define how an HTML document is displayed and how it is laid out on a page.

It is only intended to be used for styling web pages, and is not a general-purpose language like e.g javascript, lets take a look at some examples.

How can I use it?

There are multiple ways of using CSS on your webpage, only one is reccomended though, we have, Inline styles, Embedded styles, Imported styles and External styles, the only method we actualy use is External styles.

External

<html>
    <head>
        <link rel='stylesheet' href='./index.css'>
    </head>
</html>

Inline

<html>
    <body>
        <div style='background-color: #fafafa;'></div>
    </body>
</html>

Embedded

<html>
    <head>
        <style> 
            body {
                background-color: #fafafa;  
            }
        </style> 
    </head>
</html>

How css selectors work

The css selectors are used to select elements on a webpage, the syntax is,

ID selector

/* <div id='imAnElementWithAnId'></div> */                                    
#imAnElementWithAnId {
    background-color: #fafafa;  
}

Class selector

/* <div class='classElement'></div> */                                    
.classElement {
    background-color: #fafafa;  
}

Tag seletor

/* <body></body> */
body {
    background-color: #fafafa;  
}
