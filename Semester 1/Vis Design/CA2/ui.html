User interface design.

The 8 golden rules of user interface design according to Ben Shneiderman, with interactive examples.

1. Consistency

The user interface should be consistent with the rest of the application, take a look at Spotify as an example as their app is on multiple devices.

As you can see, the app is very consistent, with the same general design on all devices.

2. Shortcuts

Again, taking a look at Spotify, you can see that there are a lot of shortcuts, such as the search shortcut (Ctrl-l), the next track shortcut (Ctrl-right), and the previous track shortcut (Ctrl-left).

3. Feedback

An example of Feedback is the folder, once a user hovers over one, it becomes highlighted, once they click it, a border is shown around the folder, indicating that the user has that folder selected.

4. Dialogue

As the user performs actions that will take to execute, the user will be prompted with a dialogue box, the user want to know exactly whats happening, you want to be as descriptive as possible.
